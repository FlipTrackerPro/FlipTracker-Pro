# Packaging Integration

1. Add each supply through **FlipTracker Pro → Add Packaging Supply**.
2. Use a consistent category containing Box, Bubble, Mailer, or Tape so it appears in the appropriate sales selector.
3. Record quantities in the selected unit of measure. A tape roll may use fractional quantities such as 0.10 roll.
4. Record a sale and select the packaging used. FlipTracker Pro calculates the cost and deducts stock.
5. Rows at or below reorder level are highlighted.

Existing manually entered Packaging Cost values remain unchanged after upgrading. New sales use automatic packaging costing.
