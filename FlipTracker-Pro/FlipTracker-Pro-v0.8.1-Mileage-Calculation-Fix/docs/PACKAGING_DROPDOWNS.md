# Dynamic Packaging Dropdowns

The following Sales columns are maintained from active Packaging rows:

| Sales column | Packaging category |
|---|---|
| Box Used | Box |
| Bubble Wrap Used | Bubble Wrap |
| Mailer Used | Mailer or Envelope |
| Tape Used | Tape |
| Other Packaging Used | All other categories |

Dropdown labels use `Packaging ID — Item Name (Size)`. The stored sale record is normalized to the permanent Packaging ID when the sale form is used.

Use **FlipTracker Pro → Refresh Packaging Dropdowns** after bulk imports. Normal edits to Packaging refresh the lists automatically.
